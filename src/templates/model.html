Model Details

Diving into the Dataset

The BreakHis dataset, used in our analysis, consists of breast tumor micrographs captured at 400X magnification and contains a total of 7,909 annotated images. It is categorized into two main classes: Benign and Malignant.The Malignant class includes Invasive Ductal Carcinoma (IDC), Lobular Carcinoma (ILC), Mucinous Carcinoma, and Tubular Carcinoma. The Benign class includes tumors such as Adenoma, Fibroadenoma, Phyllodes Tumor and Tubular Adenoma.

  • The dataset comprises a total of 7,909 annotated images.
  • Includes a wide variety of images captured at 400X magnification..

Dataset Link:

https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/DBW86T

This dataset provides a comprehensive foundation for the development of machine learning models aimed at accurately classifying and detecting breast cancer in histopathological images.

Feature Extraction, Colour Space Conversion

We have uses DenseNet121, a densely connected convolutional network with 121 layers organized into four dense blocks and transition layers, to extract image features. Images are converted from RGB to HSV color space using specific formulas for Value, Saturation, and Hue to capture additional color information. Both RGB and HSV versions are processed by DenseNet121, producing 300x1000 feature vectors that are then averaged into a single unified representation.

  • DenseNet121 architecture with 121 layers and dense connectivity.
  • 300x1000 feature vector extraction from both RGB and HSV images using DenseNet121.

Principal Component Analysis (PCA)

To enable quantum classification, Principal Component Analysis (PCA) reduced the feature vector dimensionality from 300x1000 to 300x10, retaining the most significant variance and creating a compact representation compatible with qubit constraints of quantum classifiers like QSVC. This dimensionality reduction is essential for efficient mapping of data to quantum states and allows for subsequent classification using both classical (SVC) and quantum (QSVC) methods.

Compatibility with quantum hardware, reduced computational overhead, retention of key information, and noise reduction.300x10 feature matrix for subsequent classical (SVC) and quantum (QSVC) classification.

Support-Vector Classifier (SVC)

Support Vector Classifiers (SVCs) create a hyperplane to separate data points into different classes by optimizing the weight vector (w) and bias term (b) in the equation wᵀx + b = 0, using slack variables (ζᵢ) to handle non-separable data and a regularization parameter (C) to balance margin maximization and error minimization.

For this a linear kernel was chosen due to the optimized, reduced feature vector dimensions (300x10), providing an efficient solution with a linear decision boundary. This SVC application effectively utilizes features extracted from DenseNet121 to classify data, forming a base for subsequent meta-classifier and YOLO model processing.

Quantum-Support Vector Classifier (QSVC)

Quantum Support Vector Classifiers (QSVCs) utilize quantum mechanics to map data into a high-dimensional Hilbert space, offering enhanced handling of non-linearly separable data compared to classical SVCs. QSVC employs a quantum kernel for efficient projection into this space, leveraging exponentially large data representations for improved performance, especially with complex datasets.

This implementation uses the ZZFeatureMap with two circuit repetitions and linear qubit entanglement to effectively encode classical data into quantum states, capturing complex feature correlations for high-dimensional classification tasks.

Quantum-Classical Stack Ensemble

The quantum-classical stack ensemble combines predictions from classical (SVC) and quantum (QSVC) models to enhance classification performance. After training, SVC and QSVC predictions are concatenated with the original feature set (X) to create a new meta-feature set.

This augmented dataset is then used to train a meta-classifier, typically a Logistic Regression model, which learns to optimally integrate the information from both models. This approach leverages the complementary strengths of classical efficiency and quantum high-dimensionality for improved final predictions.

Custom Trained YOLOv8.

We have integrated the YOLOv8 classification model, a deep learning architecture based on the PyTorch framework, for skin disease identification on our website. This choice is driven by the model's exceptional accuracy in classifying dermatological images, making it well-suited for identifying complex patterns associated with skin diseases.

  • The model achieves top-1 accuracy of 80.65%.
  • Our model exhibits a top-5 accuracy rate of 97.2%.

By employing the YOLOv8 classification model on our website, we ensure robust skin disease identification with both top-1 and top-5 accuracy metrics, ultimately improving the accuracy and reliability of our diagnostic services.
